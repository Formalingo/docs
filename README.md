# Formalingo Documentation

Official documentation for the [Formalingo](https://formalingo.com) API — an AI-powered form builder and document-signing platform.

## Structure

| Directory | Contents |
|-----------|----------|
| `getting-started.md` | Quick start guide |
| `authentication.md` | API key authentication & scopes |
| `forms/` | Form creation, sections, questions, branding, logic |
| `documents/` | Document signing, fields, signer roles, submissions |
| `sdks/` | SDK guides — TypeScript, Python, .NET, Go |
| `webhooks/` | Webhook events & management |
| `api-reference/` | Full API endpoint reference |
| `llm.txt` | All docs concatenated for LLM consumption |

## SDKs

| SDK | Package | Install |
|-----|---------|---------|
| TypeScript | [`@formalingo/sdk`](https://www.npmjs.com/package/@formalingo/sdk) | `npm install @formalingo/sdk` |
| Python | [`formalingo-sdk`](https://github.com/Formalingo/sdk-python) | `pip install formalingo-sdk` |
| .NET | [`Formalingo.Sdk`](https://www.nuget.org/packages/Formalingo.Sdk) | `dotnet add package Formalingo.Sdk` |
| Go | [`sdk-go`](https://github.com/Formalingo/sdk-go) | `go get github.com/Formalingo/sdk-go` |
